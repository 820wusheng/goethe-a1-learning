# Goethe A1 Learning Website Builder

创建完整的Goethe-Zertifikat A1 (Start Deutsch 1) 在线学习平台

## 功能特性

- 📝 **完整试卷**: 模拟题+练习题，含听力、阅读、写作、口语4个部分
- 🎧 **双音频系统**: AI语音合成 + 官方音频链接
- 🌏 **德汉翻译**: 一键切换显示/隐藏中文翻译
- ✅ **答题功能**: 选择答案→提交→显示对错→统计分数
- 📚 **词汇学习**: 2,073个A1词汇，含例句、音频、翻译
- 🎨 **响应式设计**: 适配手机/平板/电脑

### 阶段2: 数据解析和翻译

```bash
# 1. 从PDF提取内容
pdftotext -layout <input.pdf> output.txt

# 2. 解析题目和原文
python3 scripts/parse_*.py

# 3. 人工校对翻译（强制！）
- 对照PDF官方Transkriptionen
- 修正Whisper AI错误
- 人工翻译所有内容
- 检查关键词：施奈德、19.95欧元、Zimmer 254
```

**关键点**:
- ❌ 不能直接用Whisper结果！必须手动校对
- ❌ 不能用自动翻译！必须人工翻译
- ✅ 人名保持原文或正确音译
- ✅ 数字和价格必须准确

### 阶段3: 页面开发

```bash

# 3. 配置音频
- AI朗读: speakGerman()函数
```

**关键点**:
- ✅ 双音频系统（AI + 官方）

## 历史踩坑汇总（必读！）

### 🔴 致命错误（绝对不能犯）

1. **音频直接引用外部URL** → 404错误
   - ✅ 正确：AI朗读 + 官方链接新窗口

2. **使用自动翻译** → 翻译错误
   - ✅ 正确：手动翻译所有内容

3. **跳过手动校对** → 原文错误
   - ✅ 正确：对照PDF官方文档逐字检查

4. **没有浏览器实测** → 功能异常未发现
   - ✅ 正确：自闭环测试，逐页验证

5. **跳过踩坑Review** → 重复犯错
   - ✅ 正确：开发前后都要读PITFALLS.md
