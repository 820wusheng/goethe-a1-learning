# Goethe A1 Learning Website Builder

创建完整的Goethe-Zertifikat A1 (Start Deutsch 1) 在线学习平台

## 功能特性

- 📝 **完整试卷**: 模拟题+练习题，含听力、阅读、写作、口语4个部分
- 🎧 **双音频系统**: AI语音合成 + 官方音频链接
- 🌏 **德汉翻译**: 一键切换显示/隐藏中文翻译
- ✅ **答题功能**: 选择答案→提交→显示对错→统计分数
- 📚 **词汇学习**: 2,073个A1词汇，含例句、音频、翻译
- 🎨 **响应式设计**: 适配手机/平板/电脑

## 使用方法

```
用户: 帮我创建一个Goethe A1学习网站，需要包含听力、词汇学习
```
